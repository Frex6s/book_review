# Mon vieux Grimoire


## Comment lancer le projet ? 

### Avec npm

Faites la commande `npm install` pour installer les dépendances puis `npm start` pour lancer le projet via le frontend. 

Le projet a été testé sur node 19. 

#### Le backend

Faites la commande `npm install` pour installer les dépendances

##### MangoDB

Créez une base donné et renseignez votre clé dans un fichier .env que vous devez créer

###### Lancer le backend

Faites la commande `npm start` ou `nodemon start`
